# posko-pmi-banjar

**Sistem Informasi Posko PMI Kota Banjar**

Aplikasi ini adalah sistem informasi yang digunakan untuk mengelola data dan kegiatan Posko Palang Merah Indonesia (PMI) di Kota Banjar. Dengan menggunakan PHP Native, MySQL, dan Bootstrap, aplikasi ini mempermudah pengelolaan data donor darah, laporan kegiatan, dan pemantauan posko PMI.

## Fitur Utama

- **Manajemen Posko**: Mengelola informasi posko, seperti lokasi, kontak, dan status operasional.
- **Pendataan Donor Darah**: Mencatat dan melacak informasi donor darah.
- **Laporan Kegiatan**: Mencatat dan mengelola laporan kegiatan yang dilakukan oleh PMI.
- **Pemantauan Bencana**: Memantau dan mencatat data terkait bencana yang sedang berlangsung.
- **Antarmuka Responsif**: Menggunakan **Bootstrap** untuk tampilan yang responsif dan mudah diakses di perangkat mobile maupun desktop.

## Teknologi yang Digunakan

- **Backend**: PHP Native
- **Database**: MySQL
- **Frontend**: HTML, CSS, **Bootstrap**
- **Server**: Apache
- **Authentication**: Sistem login berbasis session

## Prasyarat

Sebelum menjalankan aplikasi, pastikan Anda memiliki perangkat lunak berikut:

- PHP versi 7.4 atau lebih tinggi
- MySQL atau MariaDB
- Apache atau Nginx sebagai web server
- **Composer** (Opsional, jika menggunakan autoloader untuk PHP)
